## Project Setup
To run the program on your computer, follow the commands below and paste them into your computer's terminal:

```bash
git clone https://github.com/Dheelyte/BINCOM_DEV_TEST_2

cd BINCOM_DEV_TEST_2

python -m venv env

env\Scripts\activate # Windows
env/bin/activate # Linux / Mac

pip install django

python manage.py migrate

python manage.py runserver
```
